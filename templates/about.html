IronGate

An advanced gym access control system powered by real-time face recognition. Automatically identifies members, verifies membership status, and logs all entry attempts — no cards, no PINs, no manual check-in needed.

🎯 What This System Does

  • Identifies gym members using AI-powered face recognition (VGG-Face deep learning model)
  • Grants or denies entry based on real-time membership status verification
  • Detects emotion, gender, and age for analytics and security insights
  • Logs every entry attempt with full audit trail
  • Tracks gym occupancy in real-time (auto check-in on face recognition)
  • Records and manages membership payments (cash, GCash, card, bank)
  • Provides analytics dashboards for business intelligence
  • Manages member lifecycle: registration, renewal, freeze, suspension, deletion
  • Multi-admin system with role-based access control (superadmin / staff)
  • Complete audit log of all admin actions for accountability
  • Data backup and restore for business continuity
